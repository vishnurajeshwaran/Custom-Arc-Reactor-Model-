#  Custom Arc Reactor Model

A custom-designed **Arc Reactor replica** featuring a 3D-printed enclosure and a custom-designed PCB for LED illumination effects.

---

##  Overview

This project showcases the complete design and fabrication of a **Custom Arc Reactor Model**, including:

-  3D mechanical design and structural housing  
-  LED-based illumination system  
-  Custom-designed PCB for compact and reliable electronics integration  

The goal was to create a realistic, self-contained Arc Reactor model with a clean internal layout and professional finish.

---

##  3D Design Model

The 3D model was designed to:

- Replicate the circular, layered structure of an Arc Reactor  
- Provide internal mounting support for the PCB  
- Include LED diffusion channels for uniform light distribution  
- Ensure proper alignment of outer ring and core components  

### Key Design Elements

- **Multi-layer circular frame** for depth effect  
- **Central core housing** for LED focus  
- **PCB mounting slots/standoffs** for secure placement  
- **Wire routing paths** for clean internal cable management  
- Designed for 3D printing (PLA/resin depending on finish requirement)

The enclosure ensures even light diffusion and structural stability while keeping the electronics concealed.

---

##  PCB Design & Development

A custom PCB was created to make the lighting system compact and reliable.

### PCB Design Process

1. **Schematic Design**
   - LED array configuration
   - Current-limiting resistors
   - Power input connections
   - Switching circuitry (if included)

2. **Component Selection**
   - High-brightness LEDs
   - Proper resistor values for current control
   - Compact footprint components for circular layout

3. **PCB Layout**
   - Circular board shape to match reactor design  
   - Even LED placement for uniform glow  
   - Wide power traces for stable current flow  
   - Clean routing to minimize clutter  

4. **Fabrication & Assembly**
   - PCB fabricated from Gerber files  
   - Soldered LEDs and passive components  
   - Tested for uniform brightness and thermal safety  

---

##  Lighting Integration

- LEDs arranged symmetrically for realistic glow  
- Diffuser layer added to soften direct LED hotspots  
- Powered via external DC source or battery (based on design)
